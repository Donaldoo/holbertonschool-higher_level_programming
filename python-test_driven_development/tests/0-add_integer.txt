``add_integer`` module
======================

Using ``add_integer``, both variables must be int or float, float variables will be converted to int
-------------------

Importing:
	>>> add_integer = __import__('0-add_integer').add_integer

Add two integers:
	>>> add_integer(1, 1)
	2

Add positive and negative integers:
	>>> add_integer(1, -1)
	0

Add float and int:
	>>> add_integer(1.2, 1)
	2

Add int and string:
	>>> add_integer(1, "one")
